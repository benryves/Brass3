Manual Installation

For Windows users it is recommended that you simply use the installer package.

For other operating systems or situations where you do not have access to administrative rights you can use this zip package instead.

File Installation

Extract the files wherever you like.

For practicality reasons, all of the executables and plugins should reside in the same directory.

Environment Variables

The installer creates four environment variables:

  • Brass - directory that contains the Brass executable and plugins (%programfiles%\Brass).
  • Brass.Include - directory that contains include files (%programfiles%\Brass\Include).
  • Brass.Templates - directory that contains template files (%programfiles%\Brass\Templates).
  • Brass.Debug - directory that contains debugging files (%programfiles%\Brass\Templates).

File Associations

The following file can be merged with the Windows registry - though you'll need to modify the path to the ProjectBuilder.exe - to give Brass projects a Build context menu item.

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.brassproj]
@="Brass.Project"

[HKEY_CLASSES_ROOT\Brass.Project\shell\Build\command]
@="\"C:\\Program Files\\Brass\\ProjectBuilder.exe\" \"%1\""
